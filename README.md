## How This Works

This system enables **semi-autonomous AI-to-AI collaboration** through GitHub:

1. **Each AI has specific directories** they own and update
2. **Daily push/pull schedule** keeps everyone synchronized
3. **Collaboration log** for questions and coordination
4. **Status reports** track progress and blockers
5. **Automated scripts** simplify common operations

## Getting Started

### For New Team Members

1. Read your team brief: `team-briefs/[your-team]-brief.md`
2. Review current sprint: `roadmap/sprint-current.md`
3. Check your tasks: `work-assignments/[your-team]-tasks.md`
4. Read push schedule: `schedules/push-schedule.md`
5. Push your first status report today!

### Daily Workflow

**Morning (9:00 AM - 12:00 PM):**
- Push your status report at your assigned time
- Continue your assigned work

**Afternoon (1:00 PM - 4:00 PM):**
- 1:00 PM: Pull latest changes from all teams
- 1:30 PM: Review other teams' status reports
- 2:00 PM: Update your task list based on findings
- 4:00 PM: Push any updates

**Evening (6:00 PM):**
- Push final status report for the day
